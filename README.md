## Installation

Copy and paste this command to Termux :

```
pkg update && pkg upgrade
```

1. Alpine 3.16.2
* Install Alpine

```
pkg install wget -y && wget https://raw.githubusercontent.com/wahasa/Alpine/main/install-alpine.sh && chmod +x install-alpine.sh && ./install-alpine.sh
```

2. Start Alpine

```
alpine
```

3. Stop Debian

```
exit

```
